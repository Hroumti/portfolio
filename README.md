# Omar Hrroumti - Portfolio

A modern, responsive portfolio website showcasing my full-stack development projects and skills.

## Tech Stack

- **Frontend**: React 19, Vite, CSS3
- **Animations**: Framer Motion, Custom Particle Background
- **Contact**: EmailJS integration
- **Icons**: React Icons
- **Routing**: React Router DOM

## Features

- Responsive design with particle background animations
- Project showcase with live demos and GitHub links
- Contact form with email functionality
- Modern black & white minimalist theme
- Mobile-optimized layouts

## Quick Start

```bash
npm install
npm run dev
```

- **O-Barakat ENCG**: Educational platform with Firebase & Cloudflare
- **3D Marketplace**: React-based 3D objects marketplace
